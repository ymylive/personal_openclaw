## 一、项目背景（所有阶段通用）

### 这个项目是什么
我们要开发一个名为 `vcp-installer` 的 Rust 命令行工具，编译为单个 Windows exe 文件（约3-5MB）。

它的功能是：帮助用户在全新的 Windows 电脑上，一键完成 VCP 系统的部署。VCP 是一个 AI 智能体增强平台，由三个组件组成：

| 组件 | 是什么 | 需要什么运行时 |
|------|--------|---------------|
| VCPToolBox | 后端服务器（AI中间层） | Node.js + Python |
| VCPChat | 前端桌面客户端（Electron） | Node.js + Python |
| NewAPI | API密钥聚合管理 | 无（独立exe） |

### 核心设计理念
1. **单exe零依赖**: 用户双击exe就能用，不需要预装任何东西
2. **Portable运行时**: Git/Node.js/Python 从网上下载Portable版本，释放到安装目录的 `runtimes/` 子目录，不污染系统环境
3. **TUI交互界面**: 使用 ratatui 库实现终端图形界面，有欢迎页、环境检测、组件选择、配置表、进度条、完成页
4. **环境隔离**: 所有运行时和项目文件都在用户选择的安装目录内，自动生成启动脚本

### 最终安装目录结构
```
用户选择的安装目录/
├── VCPToolBox/          ← git clone 的后端代码
├── VCPChat/             ← git clone 的前端代码
├── runtimes/            ← Portable运行时（不污染系统）
│   ├── git/             ← PortableGit
│   ├── node/            ← Node.js portable
│   └── python/          ← Python portable
├── new-api.exe          ← NewAPI（可选）
├── start-backend.bat    ← 自动生成的启动脚本
├── start-frontend.bat   ← 自动生成的启动脚本
└── config-backup/       ← 配置备份
```

```rust
            Component::NewAPI => "NewAPI (API管理)",

        matches!(self, Component::VCPToolBox)

/// 单个安装步骤
#[derive(Debug, Clone)]
pub struct InstallStep {
    pub name: String,
    pub status: StepStatus,
    /// 如果是下载步骤，记录下载进度
    pub download_progress: Option<DownloadProgress>,
}

#[derive(Debug, Clone, PartialEq)]
pub enum StepStatus {
    Pending,      // 等待中
    Running,      // 执行中
    Completed,    // 已完成
    Failed(String), // 失败（附错误信息）
    Skipped,      // 跳过
}

#[derive(Debug, Clone)]
pub struct DownloadProgress {
    pub downloaded_bytes: u64,
    pub total_bytes: u64,
}

impl DownloadProgress {
    pub fn percentage(&self) -> f64 {
        if self.total_bytes == 0 {
            0.0
        } else {
            (self.downloaded_bytes as f64 / self.total_bytes as f64) * 100.0
        }
    }

    pub fn downloaded_mb(&self) -> f64 {
        self.downloaded_bytes as f64 / 1_048_576.0
    }

    pub fn total_mb(&self) -> f64 {
        self.total_bytes as f64 / 1_048_576.0
    }
}

/// 安装总进度
#[derive(Debug, Clone)]
pub struct InstallProgress {
    pub steps: Vec<InstallStep>,
    pub current_step_index: usize,
    pub overall_percentage: f64,
}

pub struct InstallResult {
    pub success: bool,
    pub installed_components: Vec<Component>,
    pub install_path: PathBuf,
    pub backend_start_script: Option<PathBuf>,
    pub frontend_start_script: Option<PathBuf>,
    pub errors: Vec<String>,

/// 后台安装任务发送给TUI的进度事件
#[derive(Debug, Clone)]
pub enum ProgressEvent {
    /// 某个步骤开始
    StepStarted { step_index: usize },
    /// 下载进度更新
    DownloadProgress {
        step_index: usize,
        downloaded: u64,
        total: u64,
    },
    /// 某个步骤完成
    StepCompleted { step_index: usize },
    /// 某个步骤失败
    StepFailed { step_index: usize, error: String },
    /// 某个步骤跳过
    StepSkipped { step_index: usize },
    /// 全部完成
    AllCompleted(InstallResult),
    /// 日志消息（显示在进度页底部）
    Log(String),
}

//  主应用结构体

pub struct App {
    /// 当前页面
    pub state: AppState,
    /// 是否退出
    pub should_quit: bool,
    /// 环境检测结果
    pub env_check: EnvCheckResult,
    /// 安装配置
    pub config: InstallConfig,
    /// 组件选择页的光标位置
    pub component_cursor: usize,
    /// 配置表页的光标位置
    pub config_form_cursor: usize,
    /// 配置表页的输入缓冲区（每个字段一个String）
    pub config_form_buffers: Vec<String>,
    /// 安装进度
    pub install_progress: Option<InstallProgress>,
    /// 安装结果
    pub install_result: Option<InstallResult>,
    /// 进度事件接收端（TUI在事件循环中poll这个）
    pub progress_rx: Option<mpsc::Receiver<ProgressEvent>>,
}

impl App {
    pub fn new() -> Self {
        Self {
            state: AppState::Welcome,
            should_quit: false,
            env_check: EnvCheckResult::default(),
            config: InstallConfig::default(),
            component_cursor: 0,
            config_form_cursor: 0,
            config_form_buffers: vec![
                String::new(), // install_path
                String::new(), // api_endpoint
                String::new(), // api_key
                String::new(), // admin_password
                String::new(), // tool_auth_code
                String::new(), // server_port
            ],
            install_progress: None,
            install_result: None,
            progress_rx: None,
        }
    }

    /// 切换到下一个页面
    pub fn next_page(&mut self) {
        self.state = match &self.state {
            AppState::Welcome => AppState::EnvCheck,
            AppState::EnvCheck => AppState::ComponentSelect,
            AppState::ComponentSelect => AppState::ConfigForm,
            AppState::ConfigForm => AppState::Installing,
            AppState::Installing => AppState::Complete,
            AppState::Complete => AppState::Complete, // 已是最后一页
        };
    }

    /// 切换到上一个页面
    pub fn prev_page(&mut self) {
        self.state = match &self.state {
            AppState::Welcome => AppState::Welcome,
            AppState::EnvCheck => AppState::Welcome,
            AppState::ComponentSelect => AppState::EnvCheck,
            AppState::ConfigForm => AppState::ComponentSelect,
            AppState::Installing => AppState::Installing, // 安装中不能回退
            AppState::Complete => AppState::Complete,      // 完成后不能回退
        };
    }
}
```

## 五、main.rs 完整实现

```rust
// ===== src/main.rs =====

mod app;
mod ui;
mod installer;
mod runtime;
mod utils;

use app::{App, AppState, ProgressEvent};
use anyhow::Result;
use crossterm::{
    event::{self, DisableMouseCapture, EnableMouseCapture, Event, KeyCode, KeyEventKind},
    execute,
    terminal::{disable_raw_mode, enable_raw_mode, EnterAlternateScreen, LeaveAlternateScreen},
};
use ratatui::{backend::CrosstermBackend, Terminal};
use std::io;
use std::time::Duration;

#[tokio::main]
async fn main() -> Result<()> {
    // 初始化终端
    enable_raw_mode()?;
    let mut stdout = io::stdout();
    execute!(stdout, EnterAlternateScreen, EnableMouseCapture)?;
    let backend = CrosstermBackend::new(stdout);
    let mut terminal = Terminal::new(backend)?;

    // 运行应用
    let result = run_app(&mut terminal).await;

    // 恢复终端
    disable_raw_mode()?;
    execute!(
        terminal.backend_mut(),
        LeaveAlternateScreen,
        DisableMouseCapture
    )?;
    terminal.show_cursor()?;

    if let Err(err) = result {
        eprintln!("Error: {err:?}");
    }

    Ok(())
}

async fn run_app(terminal: &mut Terminal<CrosstermBackend<io::Stdout>>) -> Result<()> {
    let mut app = App::new();

    loop {
        // 渲染当前页面
        terminal.draw(|frame| {
            match &app.state {
                AppState::Welcome => ui::welcome::render(frame, &app),
                AppState::EnvCheck => ui::env_check::render(frame, &app),
                AppState::ComponentSelect => ui::component_select::render(frame, &app),
                AppState::ConfigForm => ui::config_form::render(frame, &app),
                AppState::Installing => ui::progress::render(frame, &app),
                AppState::Complete => ui::complete::render(frame, &app),
            }
        })?;

        // 非阻塞地检查后台进度事件
        if let Some(rx) = &mut app.progress_rx {
            while let Ok(event) = rx.try_recv() {
                handle_progress_event(&mut app, event);
            }
        }

        // 等待用户输入（超时100ms以便刷新进度）
        if event::poll(Duration::from_millis(100))? {
            if let Event::Key(key) = event::read()? {
                // 只处理按下事件（忽略释放和重复）
                if key.kind != KeyEventKind::Press {
                    continue;
                }

                // 全局快捷键
                match key.code {
                    KeyCode::Char('q') | KeyCode::Char('Q') => {
                        // 安装中不允许直接退出
                        if app.state != AppState::Installing {
                            app.should_quit = true;
                        }
                    }
                    KeyCode::Esc => {
                        if app.state != AppState::Installing && app.state != AppState::Complete {
                            app.prev_page();
                        }
                    }
                    _ => {
                        // 页面级输入处理
                        handle_page_input(&mut app, key.code).await;
                    }
                }
            }
        }

        if app.should_quit {
            break;
        }
    }

    Ok(())
}

/// 处理页面级键盘输入
async fn handle_page_input(app: &mut App, key: KeyCode) {
    match &app.state {
        AppState::Welcome => {
            if matches!(key, KeyCode::Enter) {
                app.next_page();
                // TODO: P2阶段实现真正的环境检测
                // 这里暂时使用假数据
            }
        }
        AppState::EnvCheck => {
            if matches!(key, KeyCode::Enter) {
                app.next_page();
            }
        }
        AppState::ComponentSelect => {
            match key {
                KeyCode::Up => {
                    if app.component_cursor > 0 {
                        app.component_cursor -= 1;
                    }
                }
                KeyCode::Down => {
                    if app.component_cursor < 2 {
                        app.component_cursor += 1;
                    }
                }
                KeyCode::Char(' ') => {
                    // 切换组件选中状态
                    let component = match app.component_cursor {
                        0 => app::Component::VCPToolBox,
                        1 => app::Component::VCPChat,
                        2 => app::Component::NewAPI,
                        _ => return,
                    };
                    // VCPToolBox是必选的，不能取消
                    if component.is_required() {
                        return;
                    }
                    if app.config.components.contains(&component) {
                        app.config.components.retain(|c| c != &component);
                    } else {
                        app.config.components.push(component);
                    }
                }
                KeyCode::Enter => {
                    app.next_page();
                }
                _ => {}
            }
        }
        AppState::ConfigForm => {
            // TODO: P4阶段实现完整的表单交互
            if matches!(key, KeyCode::Enter) {
                app.next_page();
                // TODO: P3阶段启动真正的安装流程
            }
        }
        AppState::Installing => {
            // 安装中不处理用户输入（等待完成）
        }
        AppState::Complete => {
            match key {
                KeyCode::Enter | KeyCode::Char('q') | KeyCode::Char('Q') => {
                    app.should_quit = true;
                }
                KeyCode::Char('o') | KeyCode::Char('O') => {
                    // 打开安装目录
                    #[cfg(windows)]
                    {
                        let _ = std::process::Command::new("explorer")
                            .arg(app.config.install_path.as_os_str())
                            .spawn();
                    }
                }
                _ => {}
            }
        }
    }
}

/// 处理后台安装任务发来的进度事件
fn handle_progress_event(app: &mut App, event: ProgressEvent) {
    match event {
        ProgressEvent::StepStarted { step_index } => {
            if let Some(progress) = &mut app.install_progress {
                if let Some(step) = progress.steps.get_mut(step_index) {
                    step.status = app::StepStatus::Running;
                }
                progress.current_step_index = step_index;
            }
        }
        ProgressEvent::DownloadProgress { step_index, downloaded, total } => {
            if let Some(progress) = &mut app.install_progress {
                if let Some(step) = progress.steps.get_mut(step_index) {
                    step.download_progress = Some(app::DownloadProgress {
                        downloaded_bytes: downloaded,
                        total_bytes: total,
                    });
                }
            }
        }
        ProgressEvent::StepCompleted { step_index } => {
            if let Some(progress) = &mut app.install_progress {
                if let Some(step) = progress.steps.get_mut(step_index) {
                    step.status = app::StepStatus::Completed;
                }
                // 更新总进度百分比
                let total = progress.steps.len() as f64;
                let completed = progress.steps.iter()
                    .filter(|s| matches!(s.status, app::StepStatus::Completed | app::StepStatus::Skipped))
                    .count() as f64;
                progress.overall_percentage = (completed / total) * 100.0;
            }
        }
        ProgressEvent::StepFailed { step_index, error } => {
            if let Some(progress) = &mut app.install_progress {
                if let Some(step) = progress.steps.get_mut(step_index) {
                    step.status = app::StepStatus::Failed(error);
                }
            }
        }
        ProgressEvent::StepSkipped { step_index } => {
            if let Some(progress) = &mut app.install_progress {
                if let Some(step) = progress.steps.get_mut(step_index) {
                    step.status = app::StepStatus::Skipped;
                }
            }
        }
        ProgressEvent::AllCompleted(result) => {
            app.install_result = Some(result);
            app.state = AppState::Complete;
        }
        ProgressEvent::Log(_msg) => {
            // TODO: 在进度页底部显示日志
        }
    }
}
```

```rust
use anyhow::Result;

/// 获取Windows版本字符串
pub fn get_os_version() -> String {
    #[cfg(windows)]
    {
        use winreg::enums::HKEY_LOCAL_MACHINE;
        use winreg::RegKey;

        let hklm = RegKey::predef(HKEY_LOCAL_MACHINE);
        if let Ok(key) = hklm.open_subkey("SOFTWARE\\Microsoft\\Windows NT\\CurrentVersion") {
            let product: String = key.get_value("ProductName").unwrap_or_default();
            let build: String = key.get_value("CurrentBuildNumber").unwrap_or_default();
            return format!("{} (Build {})", product, build);
        }
        "Windows (版本未知)".to_string()
    }

    #[cfg(not(windows))]
    {
        let info = os_info::get();
        format!("{} {} {}", info.os_type(), info.version(), info.bitness())
    }
}

/// 检查磁盘剩余空间(GB)
pub fn get_available_disk_space_gb(path: &std::path::Path) -> f64 {
    use sysinfo::Disks;
    let disks = Disks::new_with_refreshed_list();

    // 找到path所在的磁盘
    let path_str = path.to_string_lossy().to_uppercase();

    for disk in disks.list() {
        let mount = disk.mount_point().to_string_lossy().to_uppercase();
        if path_str.starts_with(&mount) {
            return disk.available_space() as f64 / 1_073_741_824.0; // bytes → GB
        }
    }

    0.0
}

/// 检查是否为64位Windows
pub fn is_64bit() -> bool {
    cfg!(target_arch = "x86_64")
}
```

## 八、验收标准

完成P0后，你应该能够：

1. **`cargo build` 编译通过**，无warnings（允许unused的warning因为有空壳模块）
2. **`cargo run` 运行后**看到TUI欢迎页
3. **按Enter**可以在页面之间导航：欢迎页 → 环境检测 → 组件选择 → 配置 → 安装 → 完成
4. **按Esc**可以返回上一页
5. **按Q**可以退出程序
6. **组件选择页**可以用↑↓选择、空格切换、VCPToolBox不可取消
7. 终端退出后**正常恢复**（不会搞乱终端状态）
