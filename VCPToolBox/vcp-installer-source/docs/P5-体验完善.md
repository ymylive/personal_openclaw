```rust
/// 测试GhProxy镜像延迟
/// URL: https://ghproxy.com (HEAD)
pub async fn test_github_mirror() -> Option<Duration>

/// 镜像推荐（并发测速两个源，选延迟低的）
/// 两个都不通 → 默认GhProxy
pub async fn recommend_mirror() -> GitHubMirror

/// npm源可达性检测
/// use_mirror=false → https://registry.npmjs.org/-/ping (GET)
/// use_mirror=true  → https://registry.npmmirror.com/-/ping (GET)
pub async fn test_npm_registry(use_mirror: bool) -> bool

/// pip源可达性检测
/// use_mirror=false → https://pypi.org/simple/pip/ (GET)
/// use_mirror=true  → https://pypi.tuna.tsinghua.edu.cn/simple/pip/ (GET)
pub async fn test_pip_source(use_mirror: bool) -> bool
```

### 镜像推荐逻辑（recommend_mirror）

```rust
pub async fn recommend_mirror() -> GitHubMirror {
    let (direct, mirror) = tokio::join!(
        test_github_direct(),
        test_github_mirror()
    );
    match (direct, mirror) {
        (Some(d), Some(m)) => if d <= m { Direct } else { GhProxy },
        (Some(_), None)    => Direct,
        (None, Some(_))    => GhProxy,
        (None, None)       => GhProxy,  // 都不通默认GhProxy
    }
}
```

## 八、main.rs P5改动部分

### 1. spawn_env_check异步任务

在进入EnvCheck状态时spawn tokio任务：
- 调用platform.rs获取OS/磁盘/依赖版本
- 调用network.rs获取GitHub/npm/pip可达性
- 每个检测结果通过channel发送EnvCheckEvent
- 最后发送Done事件

### 2. env_check_rx非阻塞轮询

主循环每帧用 `try_recv()` 非阻塞检查channel，收到事件即更新app状态。

### 3. 日志滚动键盘

Progress状态下：
- `Up` → `log_scroll = log_scroll.saturating_sub(1)`
- `Down` → `log_scroll = (log_scroll + 1).min(max)`
- `PageUp` → `log_scroll = log_scroll.saturating_sub(page_size)`
- `PageDown` → `log_scroll = (log_scroll + page_size).min(max)`

### 4. Complete页O/Q键

- `Char('o') | Char('O')` → `opener::open(&install_dir)`
- `Char('q') | Char('Q') | Enter` → 退出

### 5. spinner tick计时

50ms tick驱动spinner_char()的动画帧率。
